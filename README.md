# Erremore task manager

Advanced task manager built with Next.js and NestJS. Monorepo organized with Turborepo.

## Usage

- Install dependencies:

```bash
pnpm install
```

- Run (dev):

```bash
pnpm run dev
```

- Build:

```bash
pnpm run build
```

- Lint:

```bash
pnpm run lint
```
